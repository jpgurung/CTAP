radius module¶

Used for fitting circle to the COM data of each frame of the rotating cell by 3 methods: Least Square, ODR, Algebraic Analyzes the centre of rotation and calculates the radius, meaning the distance b/w the centre of rotation and the COM

Credits: User mpastell on Github in repo SciPy-CookBook Original Code on https://github.com/mpastell/SciPy-CookBook/blob/master/originals/Least_Squares_Circle_attachments/least_squares_circle.py
